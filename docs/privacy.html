Overview

The nextjs-cicd plugin is a Claude Code skill. It provides Claude with structured knowledge and templates for setting up and auditing CI/CD pipelines for Next.js + Vercel projects. It does not collect, transmit, or store any personal data.

Data collection

This plugin does not:

  • Collect any personal information
  • Transmit data to any external server
  • Use analytics or tracking
  • Store cookies or identifiers
  • Access your file system, credentials, or environment variables directly

How the skill works

When invoked, the skill loads a SKILL.md file into Claude's context window. This file contains static rules, templates, and guidance. All processing happens locally within your Claude Code session. No data leaves your machine as a result of this plugin.

Third-party services

The skill references third-party tools (GitHub Actions, Vercel, Playwright, Vitest, Semgrep, Knip) by name for instructional purposes only. This plugin has no affiliation with or data-sharing relationship with any of these services.

Changes to this policy

If this policy changes, the updated version will be published at this URL with a revised date. The plugin itself contains no tracking mechanism, so no notification will be sent.
